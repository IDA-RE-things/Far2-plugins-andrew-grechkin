Плагин для управления сессиями Сервера терминалов
Работает только начиная с XP и выше.

Умеет:
1) Отображать список сессий
2) Отключать сессию
3) Выгружать сессию
4) Посылать сообщение
5) Отображать информацию
6) Подключаться к серверам терминалов
