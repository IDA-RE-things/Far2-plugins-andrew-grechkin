                         hdlink

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Консольная утилита для поиска дубликатов файлов и замещения их
жесткими ссылками, что позволяет экономить место на диске.

Замечание:
	Используйте на свой страх и риск.
	Автор не несет никакой ответственности за возможные
	проблемы связанные с использованием программы.

Ограничения:
	Файловая система NTFS.
	Возможно: как минимум WinXP, на 2000 не проверялось

Как работает:
1)	Составляет список всех файлов в указанных каталогах
	при этом фильтруя их в зависимости от заданных ключей
	программы.
2)	Выбрасывает из списка все файлы такого размера, который
	встречается только 1 раз.
3)	Группу файлов одинакового размера проверяет на
	- не являются ли файлы уже жесткими ссылками друг на друга
	- не находятся ли файлы на разных разделах
4)	Строится хэш SHA1 данных файлов и сравнивается для каждого
	файла. Файлы признаются идентичными при совпадении хэшей
5)	Для идентичных файлов при наличии ключа /l создаются ссылки.


Использование:
	hdlink [options] [path] [path] [...] [options]

Options:
	/? Показать краткую справку
	/l Линковать файлы, иначе просто будет проведено сравнение
	/a Атрибуты файлов обязаны совпадать
	/t Время модификации файлов обязано совпадать
	/h Пропускать скрытые фалы
	/s Пропускать системные файлы
	/j Пропускать символические ссылки (junctions)
	/m Линковать файлы меньше 1024 байта (не имеет смысла
		поскольку мелкие файлы хранятся прямо в MFT)
	/r Обходить рекурсивно все каталоги
	/q Тихий режим
	/v Многословный режим
	/d Отладочный режим

Примеры использования:
	hdlink /r /j "c:\Users"
	hdlink-x64.exe /r /j /l C:\Temp /m
